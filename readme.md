> 摒弃原Autojs框架，采用我们自主研发的DeekeScript框架；稳定性有较大提升

> 贴牌支持自定义主题（配色整体变更）

> 功能进行全面的简化，只保留用户最关心的功能

> 支持支付宝在线支付

> 支持智能评论、角色评论
